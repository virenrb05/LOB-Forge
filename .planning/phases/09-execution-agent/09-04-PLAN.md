Implement two functions in `lob_forge/executor/evaluate.py`.

**evaluate_agent(checkpoint_path, env, n_episodes=10, device="cpu") -> list[ExecutionResult]:**
- Load checkpoint from `checkpoint_path` using `torch.load(..., weights_only=False)`
- Infer obs_shape from env: `(env.seq_len, 40)`
- Instantiate `DuelingDQN(obs_shape=obs_shape, n_actions=7)` and load `checkpoint["online_net"]`
- Set to eval mode: `net.eval()`
- For each episode (seed 0 through n_episodes-1):
  - obs, info = env.reset(seed=episode_idx)
  - Greedy policy (epsilon=0): action = argmax(net(obs_tensor))
  - Collect full episode, return ExecutionResult per episode
- Return `list[ExecutionResult]`

Obs conversion: `torch.from_numpy(obs).unsqueeze(0).float().to(device)` → forward → argmax on CPU.

**compare_to_baselines(checkpoint_path, env, n_episodes=10, device="cpu") -> dict:**
- Run evaluate_agent() for DQN
- Run each baseline (TWAP, VWAP, AlmgrenChriss, Random) for n_episodes (seed 0..n_episodes-1)
- Compute mean episode_cost and mean implementation_shortfall for each
- Return dict:
```python
{
    "dqn": {"mean_cost": float, "mean_is": float, "results": list[ExecutionResult]},
    "twap": {"mean_cost": float, "mean_is": float, "results": list[ExecutionResult]},
    "vwap": {"mean_cost": float, "mean_is": float, "results": list[ExecutionResult]},
    "almgren_chriss": {"mean_cost": float, "mean_is": float, "results": list[ExecutionResult]},
    "random": {"mean_cost": float, "mean_is": float, "results": list[ExecutionResult]},
    "dqn_beats_twap": bool,  # True if dqn mean_cost < twap mean_cost
}
```
