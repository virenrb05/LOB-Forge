    **Constructor:**

    - `d_model: int = 128`
    - `channel_mults: tuple[int, ...] = (1, 2, 4, 4)`
    - `n_res_blocks: int = 2`
    - `num_timesteps: int = 1000`
    - `ddim_steps: int = 50`
    - `n_regimes: int = 3`
    - `dropout: float = 0.1`
    - `attention_levels: tuple[int, ...] = (2, 3)`
    - `n_heads: int = 4`
    - Instantiate: CosineNoiseSchedule(num_timesteps), ConditioningModule(d_model, n_regimes), UNet1D(in_channels, d_model, channel_mults, n_res_blocks, d_model, dropout, attention_levels, n_heads)

    **training_loss(x_0, regime, time_of_day=None) → Tensor:**
    - x_0: (B, T, C) clean LOB sequences — permute to (B, C, T) for U-Net
    - Sample random timesteps t ~ Uniform{0, ..., num_timesteps-1}
    - Sample noise ε ~ N(0, I) with same shape as x_0
    - Compute x_t = schedule.q_sample(x_0_perm, t, noise)
    - Get conditioning: cond = conditioning(t, regime, time_of_day)
    - Predict noise: ε_pred = unet(x_t, cond)
    - Return MSE loss: F.mse_loss(ε_pred, noise)

    **p_sample(x_t, t, cond) → Tensor:**
    - Single DDPM reverse step: x_{t-1} from x_t
    - Predict noise: ε_pred = unet(x_t, cond)
    - Compute posterior mean: μ = posterior_mean_coef1 * x_0_pred + posterior_mean_coef2 * x_t
      where x_0_pred = (x_t - sqrt(1-ᾱ_t) * ε_pred) / sqrt(ᾱ_t)
    - Add noise: x_{t-1} = μ + σ_t * z (where σ_t = sqrt(posterior_variance), z ~ N(0,I))
    - At t=0, don't add noise (return μ directly)

    **ddpm_sample(n_samples, seq_len, regime, time_of_day=None) → Tensor:**
    - Start from x_T ~ N(0, I) with shape (n_samples, in_channels, seq_len)
    - Loop t from num_timesteps-1 down to 0:
      - cond = conditioning(t_batch, regime, time_of_day) where t_batch = tensor([t]*n_samples)
      - x_t = p_sample(x_t, t_batch, cond)
    - Permute result back to (B, T, C) and return
    - Use torch.no_grad() for the entire loop
  </action>

  <action>
    Add DDIM sampling method to DiffusionModel:

    **ddim_sample(n_samples, seq_len, regime, time_of_day=None, ddim_steps=None, eta=0.0) → Tensor:**
    - If ddim_steps is None, use self.ddim_steps (default 50)
    - Create subsequence of timesteps: evenly spaced from 0 to num_timesteps-1 with ddim_steps points
      `timesteps = torch.linspace(0, self.num_timesteps - 1, ddim_steps).long()`
    - Start from x_T ~ N(0, I) with shape (n_samples, in_channels, seq_len)
    - Loop through reversed timestep pairs (t, t_prev) in the subsequence:
      - cond = conditioning(t_batch, regime, time_of_day)
      - ε_pred = unet(x_t, cond)
      - Predict x_0: x_0_pred = (x_t - sqrt(1 - ᾱ_t) * ε_pred) / sqrt(ᾱ_t)
      - Compute direction: dir = sqrt(1 - ᾱ_{t_prev} - σ²) * ε_pred
        where σ = eta * sqrt((1 - ᾱ_{t_prev}) / (1 - ᾱ_t)) * sqrt(1 - ᾱ_t / ᾱ_{t_prev})
      - x_{t_prev} = sqrt(ᾱ_{t_prev}) * x_0_pred + dir + σ * noise (if eta > 0)
    - When eta=0.0 (default), DDIM is deterministic (no noise added)
    - Permute result back to (B, T, C) and return
    - Use torch.no_grad()

    Also add a convenience method:
    **generate(n_samples, seq_len, regime, time_of_day=None, method='ddim', **kwargs) → Tensor:**
    - Routes to ddpm_sample or ddim_sample based on method string
    - Returns (B, T, C) LOB sequences
  </action>
  <verify>

import torch
from lob_forge.generator.model import DiffusionModel
model = DiffusionModel(in_channels=40, d_model=64, channel_mults=(1, 2, 4), num_timesteps=100, ddim_steps=10)
model.eval()
regime = torch.tensor([0, 1])
# DDIM sample
samples = model.ddim_sample(n_samples=2, seq_len=50, regime=regime)
assert samples.shape == (2, 50, 40), f'Expected (2, 50, 40), got {samples.shape}'
# DDIM is deterministic (eta=0)
torch.manual_seed(42)
s1 = model.ddim_sample(2, 50, regime)
torch.manual_seed(42)
s2 = model.ddim_sample(2, 50, regime)
assert torch.allclose(s1, s2), 'DDIM with eta=0 should be deterministic'
# generate() convenience
s3 = model.generate(2, 50, regime, method='ddim')
assert s3.shape == (2, 50, 40)
s4 = model.generate(2, 50, regime, method='ddpm')
assert s4.shape == (2, 50, 40)
print('DDIM + generate OK')
"
