  <name>Task 1: Implement Downsample, Upsample, and SelfAttention1D helpers</name>
  <files>lob_forge/generator/unet.py</files>
  <action>
    Create helper modules used by UNet1D:

    **Downsample1D(nn.Module):**
    - Constructor: `channels: int`
    - Use `nn.Conv1d(channels, channels, kernel_size=3, stride=2, padding=1)` — halves temporal dimension
    - Forward: `x: (B, C, T)` → `(B, C, T//2)`

    **Upsample1D(nn.Module):**
    - Constructor: `channels: int`
    - Use `nn.Upsample(scale_factor=2, mode='nearest')` followed by `nn.Conv1d(channels, channels, kernel_size=3, padding=1)` — doubles temporal dimension
    - Forward: `x: (B, C, T)` → `(B, C, T*2)`

    **SelfAttention1D(nn.Module):**
    - Constructor: `channels: int, n_heads: int = 4`
    - Use `nn.GroupNorm(min(32, channels), channels)` then `nn.MultiheadAttention(channels, n_heads, batch_first=True)`
    - Forward: `x: (B, C, T)` → transpose to `(B, T, C)` for attention → transpose back → residual connection
    - This adds global context at bottleneck resolution

    These are simple utility modules. Keep them clean and minimal.
  </action>

<task type="auto">
  <name>Task 2: Implement UNet1D encoder-decoder</name>
  <files>lob_forge/generator/unet.py</files>
  <action>
    Add UNet1D class to the same file:

    **UNet1D(nn.Module):**
    Constructor args:
    - `in_channels: int = 40` — LOB book features (generation target)
    - `d_model: int = 128` — base channel width
    - `channel_mults: tuple[int, ...] = (1, 2, 4, 4)` — channel multipliers per resolution level
    - `n_res_blocks: int = 2` — residual blocks per level
    - `cond_dim: int = 128` — conditioning vector dimension (from ConditioningModule)
    - `dropout: float = 0.1`
    - `attention_levels: tuple[int, ...] = (2, 3)` — which resolution levels get self-attention
    - `n_heads: int = 4`

    Architecture:
    1. **Input projection:** Conv1d(in_channels, d_model, 1) — lift 40 channels to d_model
    2. **Encoder:** For each level in channel_mults:
       - `n_res_blocks` × ResBlock1D(ch_in, cond_dim, out_channels=ch_out)
       - If level in attention_levels: SelfAttention1D(ch_out)
       - If not last level: Downsample1D(ch_out)
       - Store outputs in skip_connections list
    3. **Bottleneck:**
       - ResBlock1D(ch_last, cond_dim)
       - SelfAttention1D(ch_last)
       - ResBlock1D(ch_last, cond_dim)
    4. **Decoder:** For each level in reversed(channel_mults):
       - Pop skip connections and concatenate along channel dim
       - `n_res_blocks` × ResBlock1D(ch_in + ch_skip, cond_dim, out_channels=ch_out)
       - If level in attention_levels: SelfAttention1D(ch_out)
       - If not last level: Upsample1D(ch_out)
    5. **Output projection:** GroupNorm → GELU → Conv1d(d_model, in_channels, 1)

    Forward: `x: (B, C_in, T), cond: (B, cond_dim)` → `(B, C_in, T)`
    - Pass cond to every ResBlock1D
    - Return noise prediction with same shape as input

    Important: Track skip connections carefully. Each encoder level produces n_res_blocks skip tensors (one per ResBlock). The decoder must consume them in reverse order. Use a list and pop from end.

    Handle odd-length sequences: When upsampling after downsampling T=odd, the output may be T+1. Slice to match skip connection size: `x = x[..., :skip.shape[-1]]`.
  </action>
