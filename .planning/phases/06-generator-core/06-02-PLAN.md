<task type="auto">
  <name>Task 1: Implement conditioning embeddings</name>
  <files>lob_forge/generator/conditioning.py</files>
  <action>
    Create conditioning embedding modules:

    **SinusoidalTimestepEmbedding(nn.Module):**
    - Constructor: `dim: int` (output embedding dimension)
    - Forward: `t: Tensor (B,)` int64 timesteps → `(B, dim)` float32 embeddings
    - Use standard sinusoidal position encoding: half dims for sin, half for cos
    - Formula: emb = exp(-log(10000) * i / (dim/2)) where i = 0..dim/2-1, then concat(sin(t*emb), cos(t*emb))
    - Apply a 2-layer MLP after: Linear(dim, dim*4) → GELU → Linear(dim*4, dim) to learn from the fixed encodings

    **ConditioningModule(nn.Module):**
    - Constructor: `d_model: int = 128, n_regimes: int = 3, time_of_day_dim: int = 16`
    - Components:
      - `timestep_emb`: SinusoidalTimestepEmbedding(d_model)
      - `regime_emb`: nn.Embedding(n_regimes, d_model)
      - `tod_mlp`: nn.Sequential(Linear(1, time_of_day_dim), GELU, Linear(time_of_day_dim, d_model)) — maps scalar time-of-day ∈ [0, 1] to d_model
    - Forward: `t: Tensor (B,), regime: Tensor (B,), time_of_day: Tensor (B,) or None` → `(B, d_model)`
    - Sum all embeddings: `cond = timestep_emb(t) + regime_emb(regime) + tod_emb(time_of_day)`
    - If time_of_day is None, skip that term (make it optional for inference flexibility)
    - Return the combined conditioning vector
  </action>
  <verify>
    python -c "
import torch
from lob_forge.generator.conditioning import SinusoidalTimestepEmbedding, ConditioningModule
# Timestep embedding
te = SinusoidalTimestepEmbedding(128)
t = torch.arange(0, 1000, 100)
emb = te(t)
assert emb.shape == (10, 128)
# Different timesteps produce different embeddings
assert not torch.allclose(emb[0], emb[1])
# Conditioning module
cm = ConditioningModule(d_model=128, n_regimes=3)
t = torch.tensor([0, 500, 999])
regime = torch.tensor([0, 1, 2])
tod = torch.tensor([0.0, 0.5, 1.0])
cond = cm(t, regime, tod)
assert cond.shape == (3, 128)
# Without time_of_day
cond2 = cm(t, regime, time_of_day=None)
assert cond2.shape == (3, 128)
print('conditioning OK')
"
  </verify>

<task type="auto">
  <name>Task 2: Implement AdaLN and ResBlock1D</name>
  <files>lob_forge/generator/blocks.py</files>
  <action>
    Create the core building blocks for the 1D U-Net:

    **AdaptiveLayerNorm(nn.Module):**
    - Constructor: `channels: int, cond_dim: int`
    - Contains: `norm = nn.GroupNorm(num_groups=32, num_channels=channels)` (use min(32, channels) for group count)
    - Contains: `proj = nn.Linear(cond_dim, channels * 2)` — projects conditioning to scale and shift
    - Forward: `x: (B, C, T), cond: (B, cond_dim)` → `(B, C, T)`
    - Compute: `scale, shift = proj(cond).chunk(2, dim=-1)` → each (B, C)
    - Apply: `x = norm(x) * (1 + scale.unsqueeze(-1)) + shift.unsqueeze(-1)`
    - The `1 + scale` is important — initializes as identity transform

    **ResBlock1D(nn.Module):**
    - Constructor: `channels: int, cond_dim: int, dropout: float = 0.1, out_channels: int | None = None`
    - If out_channels is None, use channels (same in/out)
    - Architecture:
      1. AdaLN(channels, cond_dim) → GELU → Conv1d(channels, out_channels, kernel_size=3, padding=1)
      2. AdaLN(out_channels, cond_dim) → GELU → Dropout → Conv1d(out_channels, out_channels, kernel_size=3, padding=1)
      3. Skip connection: if channels != out_channels, use Conv1d(channels, out_channels, 1) for projection; else identity
    - Forward: `x: (B, C, T), cond: (B, cond_dim)` → `(B, out_C, T)`
    - Return: skip(x) + block(x)

    Initialize Conv1d weights with Xavier uniform and biases to zero for stable training.
  </action>
  <verify>
    python -c "
import torch
from lob_forge.generator.blocks import AdaptiveLayerNorm, ResBlock1D
# AdaLN
aln = AdaptiveLayerNorm(128, cond_dim=128)
x = torch.randn(2, 128, 50)
cond = torch.randn(2, 128)
out = aln(x, cond)
assert out.shape == (2, 128, 50)
# ResBlock same channels
rb = ResBlock1D(128, cond_dim=128)
out = rb(x, cond)
assert out.shape == (2, 128, 50)
# ResBlock different channels
rb2 = ResBlock1D(128, cond_dim=128, out_channels=256)
out2 = rb2(x, cond)
assert out2.shape == (2, 256, 50)
print('blocks OK')
"
  </verify>
  <done>AdaLN applies conditioning-derived scale/shift; ResBlock1D uses AdaLN with skip connections; channel changes handled</done>
