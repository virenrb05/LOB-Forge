<task type="auto">
  <name>Task 1: Implement resampling and rolling z-score normalization</name>
  <files>lob_forge/data/preprocessor.py</files>
  <action>
    Replace the stub preprocessor.py with two core functions:

    1. resample_to_grid(df, interval_ms=100) → pd.DataFrame:
       - Convert timestamp (int64 microseconds) to pandas DatetimeIndex
       - Resample to uniform grid at interval_ms using forward-fill (ffill)
       - Log fill rate: (original_rows / resampled_rows) as info message using Python logging
       - Convert back to int64 microseconds timestamps
       - Return resampled DataFrame

    2. rolling_zscore(df, feature_cols, window=5000) → tuple[pd.DataFrame, pd.DataFrame]:
       - For each column in feature_cols: z = (x - rolling_mean) / rolling_std
       - Use df[col].rolling(window, min_periods=1).mean() and .std() — CAUSAL (backward-looking only)
       - Handle zero std: replace with 1.0 to avoid division by zero
       - First `window` rows will have partial statistics — this is expected (use min_periods=1)
       - Return (normalized_df, stats_df) where stats_df has columns {col}_mean and {col}_std for inverse transform
       - stats_df enables the generator to denormalize outputs later

    Import logging at module level. Use schema.py column constants to identify which columns are features vs metadata.
  </action>
  <verify>

<task type="auto">
  <name>Task 2: Implement full preprocessing pipeline</name>
  <files>lob_forge/data/preprocessor.py, configs/data.yaml</files>
  <action>
    Add the main pipeline function to preprocessor.py:

    preprocess(input_path, output_dir, cfg) → dict:
    - input_path: Path to raw LOB Parquet file
    - output_dir: Path to save processed files
    - cfg: dict or OmegaConf with keys: resample_interval_ms, horizons, label_threshold, normalization_window, split.train, split.val, split.test, purge_gap
    - Pipeline steps:
      1. df = read_lob_parquet(input_path)
      2. df = resample_to_grid(df, cfg.resample_interval_ms)
      3. df = compute_all_features(df, lookbacks=[1, 5, 10, 50])
      4. df = compute_labels(df, horizons=cfg.horizons, threshold=cfg.label_threshold)
      5. Identify feature columns (all numeric cols except timestamp, label_*, trade_side)
      6. df_normed, stats = rolling_zscore(df, feature_cols, window=cfg.normalization_window)
      7. train_idx, val_idx, test_idx = temporal_split(len(df_normed), ratios=(cfg.split.train, cfg.split.val, cfg.split.test), purge_gap=cfg.get('purge_gap', 0))
      8. Save df_normed.iloc[train_idx] → output_dir/train.parquet
      9. Save df_normed.iloc[val_idx] → output_dir/val.parquet
      10. Save df_normed.iloc[test_idx] → output_dir/test.parquet
      11. Save stats → output_dir/normalization_stats.parquet
      12. Return dict with row counts per split and file paths

    Also update configs/data.yaml to add `purge_gap: 100` (100 rows = 10 seconds at 100ms).

    Use write_lob_parquet for the raw LOB columns, but for the augmented preprocessed data use plain pq.write_table since it has extra feature/label columns beyond the LOB schema.
  </action>
