  <action>
    Create `lob_forge/evaluation/lob_bench.py` with:

    1. **compute_wasserstein_metrics(real: np.ndarray, synthetic: np.ndarray) -> dict[str, float]**
       - real, synthetic: (N, C) arrays where C = number of features (typically 40 LOB columns)
       - Compute 1D Wasserstein distance (scipy.stats.wasserstein_distance) per feature column
       - Return dict with:
         - "wd_mean": mean across all features
         - "wd_max": max across features
         - "wd_per_feature": list of per-feature distances
         - Also compute on derived quantities:
           - "wd_spread": Wasserstein on spread series (ask_1 - bid_1)
           - "wd_mid": Wasserstein on mid-price series ((ask_1 + bid_1) / 2)
           - "wd_returns": Wasserstein on log-return series

    2. **compute_conditional_stats(real: np.ndarray, synthetic: np.ndarray, real_regimes: np.ndarray, synthetic_regimes: np.ndarray) -> dict[str, dict]**
       - For each regime (0, 1, 2), compute within-regime statistics:
         - Mean and std of spread
         - Mean and std of total book depth (sum of all sizes)
         - Mean and std of mid-price returns
       - Compare real vs synthetic within each regime using relative error: |real - synth| / (|real| + eps)
       - Return dict keyed by regime → {spread_mean_err, depth_mean_err, return_std_err, ...}
       - Also return "mean_relative_error": average across all regime-stat pairs

    Add `__all__`, docstrings, type hints. Use numpy and scipy only for these two functions.
  </action>
