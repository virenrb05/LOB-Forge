    Create `lob_forge/evaluation/regime_validation.py` with:

    1. **compare_regime_distributions(data_by_regime: dict[int, np.ndarray]) -> dict[str, dict]**
       - data_by_regime: {0: (N0, C), 1: (N1, C), 2: (N2, C)} — LOB data grouped by regime
       - For each pair of regimes (0-1, 0-2, 1-2), compute:
         - KS test on mid-price returns (derived from columns 0 and 20: (ask_1 + bid_1) / 2)
         - KS test on spread distributions
         - Ratio of return standard deviations
       - Return dict keyed by regime pair → {ks_returns_stat, ks_returns_p, ks_spread_stat, ks_spread_p, vol_ratio}

    2. **compute_regime_divergence(data_by_regime: dict[int, np.ndarray], n_bins=50) -> dict[str, float]**
       - Compute discrete KL divergence between regime return distributions
       - Histogram returns for each regime, add epsilon for stability
       - Return: kl_01, kl_02, kl_12 (KL divergence for each pair)
       - Also return: mean_kl (average divergence), regime_separability (True if mean_kl > 0.1)

    3. **validate_regime_conditioning(real_by_regime: dict[int, np.ndarray], synthetic_by_regime: dict[int, np.ndarray]) -> dict[str, Any]**
       - Orchestrator: compares both real and synthetic regime distributions
       - Checks:
         a. Synthetic regimes are distinct from each other (compare_regime_distributions on synthetic)
         b. Each synthetic regime matches its corresponding real regime (KS test per regime)
         c. Regime ordering preserved: vol(regime=2) > vol(regime=1) > vol(regime=0) in synthetic
       - Return: regime_distinct (bool), regime_matched (bool), ordering_preserved (bool), all_passed (bool), details (nested dict)

    Use numpy, scipy.stats. Add `__all__`, docstrings, type hints.
