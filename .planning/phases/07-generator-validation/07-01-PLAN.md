  <name>Task 1: Implement return and volatility stylized fact tests</name>
  <files>lob_forge/evaluation/stylized_facts.py</files>
  <action>
    Replace the stub with a module implementing three functions. All functions follow the same signature pattern:
    `def test_name(real: np.ndarray, synthetic: np.ndarray, **kwargs) -> dict[str, float | bool]`

    Each returns a dict with at minimum: `{"statistic": float, "p_value": float, "passed": bool}`.
    Default threshold for "passed" is p_value > 0.05 (configurable via `alpha` kwarg).

    1. **return_distribution_test(real_mid, synthetic_mid, alpha=0.05)**
       - Compute log-returns from mid-price series: `np.diff(np.log(mid))`
       - Two-sample KS test (scipy.stats.ks_2samp) on returns
       - Return: statistic, p_value, passed, plus kurtosis of both (real_kurtosis, synthetic_kurtosis)
       - Both should show excess kurtosis (fat tails)

    2. **volatility_clustering_test(real_mid, synthetic_mid, max_lag=20)**
       - Compute absolute log-returns for both series
       - Compute autocorrelation of |returns| at lags 1..max_lag using np.correlate (normalized)
       - Return: mean_acf_real, mean_acf_synthetic, acf_ratio (synthetic/real), passed
       - Pass if mean ACF of |returns| for synthetic is > 0 (shows clustering) AND acf_ratio in [0.3, 3.0]

    3. **bid_ask_bounce_test(real_returns, synthetic_returns)**
       - Compute lag-1 autocorrelation of the return series
       - Both should be negative (bid-ask bounce causes negative autocorrelation at short lags)
       - Return: real_lag1_acf, synthetic_lag1_acf, passed
       - Pass if both autocorrelations are negative

    Use numpy and scipy only. Add module-level docstring. Add `__all__` list.
    Import scipy.stats for ks_2samp, kurtosis. Import numpy.
  </action>

       - Concavity means beta < 1.0 (square-root law of market impact)
       - Return: real_beta, synthetic_beta, passed
       - Pass if both betas are < 1.0 and both are > 0 (positive but concave)
