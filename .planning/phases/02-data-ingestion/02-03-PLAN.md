    Create `lob_forge/data/lobster.py` with `LOBSTERAdapter` class:

    **Constructor:**
    ```
    __init__(self, depth=10, output_dir='data/')
    ```

    **LOBSTER file format (reference):**
    - Two files per ticker-day:
      - `{TICKER}_{date}_{start}_{end}_orderbook_{N}.csv` — orderbook snapshots
      - `{TICKER}_{date}_{start}_{end}_message_{N}.csv` — event messages
    - Orderbook file: NO header. Columns alternate: ask_price_1, ask_size_1, bid_price_1, bid_size_1, ask_price_2, ask_size_2, bid_price_2, bid_size_2, ...
      - Note: LOBSTER uses ask-first ordering and alternates ask/bid per level
      - Prices are in dollar-cents (integer, divide by 10000 to get dollars)
    - Message file: NO header. Columns: time (seconds since midnight, float64), event_type, order_id, size, price, direction
      - event_type: 1=new, 2=partial cancel, 3=full cancel, 4=execution (visible), 5=execution (hidden), 6=cross trade, 7=halt
      - direction: -1=sell, 1=buy

    **Method: `convert_file(orderbook_path: Path, message_path: Path, output_path: Path | None = None) -> Path`**
    1. Read message file to get timestamps (column 0 = time in seconds since midnight as float64)
    2. Read orderbook file (all columns numeric, no header)
    3. Determine source depth N from orderbook column count: N = num_cols / 4
    4. Transpose columns from LOBSTER ordering to unified schema:
       - LOBSTER col 0 = ask_price_1, col 1 = ask_size_1, col 2 = bid_price_1, col 3 = bid_size_1
       - LOBSTER col 4 = ask_price_2, col 5 = ask_size_2, col 6 = bid_price_2, col 7 = bid_size_2
       - General: level i (1-indexed): ask_price at col (i-1)*4, ask_size at (i-1)*4+1, bid_price at (i-1)*4+2, bid_size at (i-1)*4+3
    5. If source depth > target depth (10): truncate to first 10 levels
       If source depth < target depth: pad missing levels with NaN
    6. Convert LOBSTER prices: divide by 10000 (LOBSTER stores as integer cents × 100)
    7. Compute derived columns:
       - timestamp: convert message file time (seconds since midnight) to unix microseconds.
         Require a `date` parameter or extract from filename to compute full timestamp.
       - mid_price = (bid_price_1 + ask_price_1) / 2
       - spread = ask_price_1 - bid_price_1
       - trade_price/trade_size/trade_side: extract from message file where event_type in [4, 5] (executions). For non-execution rows, set to NaN/NaN/0.
    8. Construct DataFrame with ALL_COLUMNS ordering
    9. Write using `write_lob_parquet`

    **Method: `convert_directory(input_dir: Path, output_dir: Path | None = None) -> list[Path]`**
    - Scan input_dir for LOBSTER file pairs (match orderbook + message files by ticker/date)
    - Convert each pair, collect output paths
    - Return list of output Parquet paths

    **Error handling:**
    - Raise ValueError if orderbook/message row counts don't match
    - Warn if depth < 10 (padding with NaN)
    - Skip malformed rows, log count of skipped rows
  </action>

    from lob_forge.data.lobster import LOBSTERAdapter
    import pandas as pd, numpy as np
    from pathlib import Path
    import tempfile, os

    # Create synthetic LOBSTER files for testing
    tmpdir = tempfile.mkdtemp()

    # 3 levels, 2 rows: columns = ask_p1, ask_s1, bid_p1, bid_s1, ask_p2, ask_s2, bid_p2, bid_s2, ask_p3, ask_s3, bid_p3, bid_s3
    ob_data = np.array([
        [1000100, 50, 999900, 30, 1000200, 40, 999800, 20, 1000300, 35, 999700, 15],
        [1000150, 55, 999950, 35, 1000250, 45, 999850, 25, 1000350, 38, 999750, 18],
    ])
    msg_data = np.array([
        [34200.001, 1, 1001, 10, 1000100, 1],
        [34200.002, 4, 1002, 5, 999900, -1],
    ])
    np.savetxt(f'{tmpdir}/TEST_2024-01-01_34200000_57600000_orderbook_3.csv', ob_data, delimiter=',', fmt='%d')
    np.savetxt(f'{tmpdir}/TEST_2024-01-01_34200000_57600000_message_3.csv', msg_data, delimiter=',', fmt='%.6f')

    adapter = LOBSTERAdapter(depth=10, output_dir=tmpdir)
    out = adapter.convert_file(
        Path(f'{tmpdir}/TEST_2024-01-01_34200000_57600000_orderbook_3.csv'),
        Path(f'{tmpdir}/TEST_2024-01-01_34200000_57600000_message_3.csv'),
        date='2024-01-01'
    )
    df = pd.read_parquet(out)
    print(f'Output shape: {df.shape}')
    print(f'Columns: {len(df.columns)}')
    print(f'bid_price_1: {df.bid_price_1.iloc[0]:.4f}')
    print(f'ask_price_1: {df.ask_price_1.iloc[0]:.4f}')
    # Levels 4-10 should be NaN since source only has 3 levels
    print(f'bid_price_4 is NaN: {pd.isna(df.bid_price_4.iloc[0])}')
    "
