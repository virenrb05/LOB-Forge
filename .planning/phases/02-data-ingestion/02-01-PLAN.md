<task type="auto">
  <name>Task 2: Create data validation module</name>
  <files>lob_forge/data/validation.py, lob_forge/data/__init__.py</files>
  <action>
    Create `lob_forge/data/validation.py` with integrity check functions:

    1. `validate_lob_dataframe(df: pd.DataFrame) -> list[str]`:
       - Returns list of warning/error strings (empty = valid)
       - Checks performed:
         a. No NaN values in price/size columns (trade columns may have NaN)
         b. All prices positive (bid_price_*, ask_price_* > 0)
         c. All sizes non-negative (bid_size_*, ask_size_* >= 0)
         d. No crossed books: bid_price_1 <= ask_price_1 for every row
         e. Monotonically increasing timestamps
         f. Timestamp gaps > 500ms logged as warnings (not errors)
         g. Bid prices descending: bid_price_1 >= bid_price_2 >= ... >= bid_price_10
         h. Ask prices ascending: ask_price_1 <= ask_price_2 <= ... <= ask_price_10

    2. `compute_quality_metrics(df: pd.DataFrame) -> dict`:
       - Returns dict with: row_count, time_span_seconds, mean_gap_ms, max_gap_ms, gap_count_over_500ms, crossed_book_count, nan_count

    Import key symbols in `lob_forge/data/__init__.py`:
    ```python
    from lob_forge.data.schema import LOB_SCHEMA, ALL_COLUMNS, write_lob_parquet, read_lob_parquet
    from lob_forge.data.validation import validate_lob_dataframe, compute_quality_metrics
    ```
