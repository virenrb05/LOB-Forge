<task type="auto">
  <name>Task 1: Implement FocalLoss</name>
  <files>lob_forge/predictor/losses.py</files>
  <action>
    Create new file `lob_forge/predictor/losses.py` with `FocalLoss(nn.Module)`.

    Constructor parameters:
    - `gamma: float = 2.0` — focusing parameter (higher = more focus on hard examples)
    - `class_weights: Tensor | None = None` — per-class weights for imbalance (e.g., inverse frequency)
    - `reduction: str = "mean"` — "mean", "sum", or "none"

    Implementation (from scratch — do NOT use external library):
    1. Compute log-softmax: `log_p = F.log_softmax(logits, dim=-1)`
    2. Gather log-prob for true class: `log_pt = log_p.gather(dim=-1, index=targets.unsqueeze(-1)).squeeze(-1)`
    3. Compute pt: `pt = log_pt.exp()`
    4. Compute focal weight: `focal_weight = (1 - pt) ** gamma`
    5. Apply class weights if provided: `if class_weights is not None: w = class_weights[targets]; focal_weight = focal_weight * w`
    6. Loss: `loss = -focal_weight * log_pt`
    7. Apply reduction

    Forward signature: `forward(self, logits: Tensor, targets: Tensor) -> Tensor`
    - logits: (batch, n_classes) or (batch, n_horizons, n_classes)
    - targets: (batch,) or (batch, n_horizons) int64
    - If 3D logits, flatten first two dims, compute loss, reshape back, then reduce

    Mathematical verification: when gamma=0 and no class weights, focal loss = cross-entropy.
    When gamma>0, focal loss down-weights well-classified examples (pt close to 1).

    Register class_weights as a buffer (not parameter) so it moves with .to(device).
  </action>
  <verify>
    ```python
    python -c "
    import torch
    import torch.nn.functional as F
    from lob_forge.predictor.losses import FocalLoss

    # Test 1: gamma=0 should match cross-entropy
    logits = torch.randn(8, 3)
    targets = torch.randint(0, 3, (8,))
    fl = FocalLoss(gamma=0.0)
    ce = F.cross_entropy(logits, targets)
    assert torch.allclose(fl(logits, targets), ce, atol=1e-5), 'gamma=0 should match CE'

    # Test 2: gamma=2 should produce lower loss for confident predictions
    fl2 = FocalLoss(gamma=2.0)
    loss_g0 = fl(logits, targets)
    loss_g2 = fl2(logits, targets)
    # Focal loss <= CE (it down-weights easy examples)
    assert loss_g2 <= loss_g0 + 1e-5, 'Focal should be <= CE'

    # Test 3: 3D input (multi-horizon)
    logits_3d = torch.randn(8, 4, 3)
    targets_3d = torch.randint(0, 3, (8, 4))
    loss_3d = fl2(logits_3d, targets_3d)
    assert loss_3d.dim() == 0, 'Should be scalar with mean reduction'

    print('FocalLoss: OK (gamma=0 matches CE, 3D input works)')
    "
    ```
  </verify>
  <done>FocalLoss implemented from scratch. gamma=0 matches cross-entropy exactly. Supports class weights (as buffer), 2D and 3D logits, configurable reduction.</done>

<task type="auto">
  <name>Task 2: Implement DualAttentionTransformer</name>
  <files>lob_forge/predictor/model.py</files>
  <action>
    Replace the stub in `lob_forge/predictor/model.py` with the full `DualAttentionTransformer(nn.Module)`.

    Constructor parameters (all from predictor.yaml):
    - `n_levels: int = 10` — LOB levels
    - `features_per_level: int = 4` — features per level
    - `d_model: int = 64` — embedding dimension
    - `n_heads: int = 4`
    - `n_spatial_layers: int = 2`
    - `n_temporal_layers: int = 2`
    - `feedforward_dim: int = 128`
    - `dropout: float = 0.1`
    - `n_classes: int = 3`
    - `n_horizons: int = 4` — number of prediction horizons
    - `max_seq_len: int = 512` — max sequence length for positional encoding
    - `vpin_head: bool = True` — whether to include VPIN regression head

    Architecture (per tech spec Section 3.2):

    **1. Input Embedding:**
    - Input: (batch, T, n_levels * features_per_level) = (batch, T, 40)
    - Reshape to (batch, T, n_levels, features_per_level) = (batch, T, 10, 4)
      Column ordering in schema: bid_price_1..10, bid_size_1..10, ask_price_1..10, ask_size_1..10.
      So flat features are 4 groups of 10. Reshape via `.view(B, T, 4, 10).permute(0, 1, 3, 2)` → (B, T, 10, 4)
    - Project: `nn.Linear(features_per_level, d_model)` → (batch, T, 10, d_model)
    - Add learnable positional encodings:
      - `level_pe = nn.Parameter(torch.randn(1, 1, n_levels, d_model) * 0.02)` — broadcast over batch and T
      - `time_pe = nn.Parameter(torch.randn(1, max_seq_len, 1, d_model) * 0.02)` — broadcast over batch and L
      - Sum: `x = x + level_pe + time_pe[:, :T, :, :]`

    **2. Spatial Attention:**
    - Reshape (batch, T, L, d_model) → (batch*T, L, d_model)
    - Pass through `SpatialAttentionBlock(d_model, n_heads, n_spatial_layers, feedforward_dim, dropout)`
    - Reshape back to (batch, T, L, d_model)

    **3. Level Pooling:**
    - Mean pool across levels: (batch, T, L, d_model) → (batch, T, d_model)

    **4. Temporal Attention:**
    - Pass through `TemporalAttentionBlock(d_model, n_heads, n_temporal_layers, feedforward_dim, dropout, max_seq_len)`
    - Output: (batch, T, d_model)

    **5. Classification Heads (per horizon):**
    - Take last time step: (batch, d_model)
    - Per-horizon head: `nn.ModuleList` of `n_horizons` heads, each:
      `nn.Sequential(nn.LayerNorm(d_model), nn.Linear(d_model, d_model), nn.GELU(), nn.Dropout(dropout), nn.Linear(d_model, n_classes))`
    - Stack outputs: (batch, n_horizons, n_classes)

    **6. VPIN Head (optional):**
    - If `vpin_head=True`: `nn.Sequential(nn.LayerNorm(d_model), nn.Linear(d_model, d_model // 2), nn.GELU(), nn.Linear(d_model // 2, 1), nn.Sigmoid())`
    - Takes same last-timestep embedding as classification heads
    - Output: (batch, 1) — VPIN estimate in [0, 1]

    Forward returns: `dict` with keys:
    - `"logits"`: (batch, n_horizons, n_classes) — classification logits
    - `"vpin"`: (batch, 1) — VPIN prediction (only if vpin_head=True)
    - `"embedding"`: (batch, d_model) — last-timestep embedding (for downstream generator conditioning in Phase 6)

    Import SpatialAttentionBlock from spatial_attention and TemporalAttentionBlock from temporal_attention.
  </action>
  <verify>
    ```python
    python -c "
    import torch
    from lob_forge.predictor.model import DualAttentionTransformer

    m = DualAttentionTransformer(
        n_levels=10, features_per_level=4, d_model=64, n_heads=4,
        n_spatial_layers=2, n_temporal_layers=2, feedforward_dim=128,
        dropout=0.1, n_classes=3, n_horizons=4, vpin_head=True,
    )
    x = torch.randn(16, 100, 40)
    out = m(x)
    assert out['logits'].shape == (16, 4, 3), f'logits: {out[\"logits\"].shape}'
    assert out['vpin'].shape == (16, 1), f'vpin: {out[\"vpin\"].shape}'
    assert out['embedding'].shape == (16, 64), f'embedding: {out[\"embedding\"].shape}'
    assert (out['vpin'] >= 0).all() and (out['vpin'] <= 1).all(), 'VPIN not in [0,1]'

    # Without VPIN head
    m2 = DualAttentionTransformer(n_levels=10, features_per_level=4, vpin_head=False)
    out2 = m2(x)
    assert 'vpin' not in out2, 'vpin should not be in output when disabled'

    params = sum(p.numel() for p in m.parameters())
    print(f'DualAttentionTransformer: OK — {params:,} params')
    print(f'  logits: {out[\"logits\"].shape}, vpin: {out[\"vpin\"].shape}, emb: {out[\"embedding\"].shape}')
    "
    ```
  </verify>
  <done>DualAttentionTransformer composes spatial + temporal attention with per-horizon classification heads and optional VPIN head. Returns dict with logits, vpin, embedding. Input: (batch, T, 40), Output: logits (batch, 4, 3), vpin (batch, 1), embedding (batch, 64).</done>
