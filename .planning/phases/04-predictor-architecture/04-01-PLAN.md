<objective>
Implement the two attention building blocks for the TLOB dual-attention transformer: spatial attention across LOB levels and temporal attention across time steps.

Purpose: These are the core differentiating components of the TLOB architecture (arXiv 2502.15757). Spatial attention captures cross-level relationships (e.g., level-1 imbalance vs level-5 depth), while temporal attention with causal masking captures sequential order flow patterns.

Output: Two standalone nn.Module classes ready for composition in the DualAttentionTransformer.
</objective>

  <action>
    Create `SpatialAttentionBlock(nn.Module)` that applies multi-head self-attention across LOB levels for each time step independently.

    Constructor parameters:
    - `d_model: int = 64` — embedding dimension
    - `n_heads: int = 4` — attention heads
    - `n_layers: int = 2` — number of transformer encoder layers
    - `feedforward_dim: int = 128` — FFN hidden dimension
    - `dropout: float = 0.1`

    Architecture:
    - Use `nn.TransformerEncoder` wrapping `nn.TransformerEncoderLayer(d_model, n_heads, feedforward_dim, dropout, activation='gelu', batch_first=True, norm_first=True)`.
    - `norm_first=True` (Pre-LN) for stable training — this is the modern default and avoids the learning rate sensitivity of Post-LN.

    Forward signature: `forward(self, x: Tensor) -> Tensor`
    - Input: `(batch * T, L, d_model)` where L = number of LOB levels
    - Output: same shape `(batch * T, L, d_model)`
    - The caller (DualAttentionTransformer) handles reshaping from `(batch, T, L, d_model)` to `(batch*T, L, d_model)` before calling this module.

    Include module docstring explaining the spatial attention concept: for each time step, the model learns cross-level relationships in the order book.
  </action>
  <verify>

    x = torch.randn(32 * 100, 10, 64)  # batch*T=3200, L=10, d_model=64
    y = m(x)

<task type="auto">
  <name>Task 2: Implement TemporalAttentionBlock</name>
  <files>lob_forge/predictor/temporal_attention.py</files>
  <action>
    Create `TemporalAttentionBlock(nn.Module)` that applies multi-head self-attention across time steps with causal masking.

    Constructor parameters:
    - `d_model: int = 64`
    - `n_heads: int = 4`
    - `n_layers: int = 2`
    - `feedforward_dim: int = 128`
    - `dropout: float = 0.1`
    - `max_seq_len: int = 512` — maximum sequence length for mask precomputation

    Architecture:
    - Use `nn.TransformerEncoder` wrapping `nn.TransformerEncoderLayer(d_model, n_heads, feedforward_dim, dropout, activation='gelu', batch_first=True, norm_first=True)`.
    - Generate a causal (upper-triangular) attention mask so each time step t only attends to steps 0..t. Use `torch.nn.Transformer.generate_square_subsequent_mask(max_seq_len)` and register as a buffer. In forward(), slice the mask to the actual sequence length T.
    - The mask must be boolean (True = masked positions) for PyTorch >= 2.0 `is_causal` style, OR use float -inf mask. Use `torch.triu(torch.ones(T, T, dtype=torch.bool), diagonal=1)` registered as buffer, sliced in forward.

    Forward signature: `forward(self, x: Tensor) -> Tensor`
    - Input: `(batch, T, d_model)` — after spatial attention and level pooling
    - Output: same shape `(batch, T, d_model)`
    - Slice the precomputed mask to `[:T, :T]` and pass as `src_mask` to the encoder.

    Include docstring explaining causal temporal attention: the model captures sequential patterns in order flow while preventing information leakage from future time steps.
  </action>
  <verify>
    ```python
    python -c "
    import torch
    from lob_forge.predictor.temporal_attention import TemporalAttentionBlock
    m = TemporalAttentionBlock(d_model=64, n_heads=4, n_layers=2, feedforward_dim=128)
    x = torch.randn(32, 100, 64)  # batch=32, T=100, d_model=64
    y = m(x)
    assert y.shape == (32, 100, 64), f'Expected (32,100,64), got {y.shape}'
    # Verify causal masking: output at t=0 should not change when future inputs change
    x2 = x.clone()
    x2[:, 50:, :] = torch.randn(32, 50, 64)  # change future
    y2 = m(x2)
    assert torch.allclose(y[:, 0, :], y2[:, 0, :], atol=1e-5), 'Causal violation: t=0 changed'
    print('TemporalAttentionBlock: OK (causal masking verified)')
    "
    ```
  </verify>
  <done>TemporalAttentionBlock accepts (batch, T, d_model), returns same shape, and enforces causal masking (verified: modifying future inputs does not affect past outputs).</done>
</task>
