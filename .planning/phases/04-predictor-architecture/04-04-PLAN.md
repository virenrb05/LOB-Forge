  <action>
    Create `tests/test_predictor.py` with thorough tests for all predictor components. Follow the existing test style in `tests/test_features.py` (pytest, class-based organization).

    **Test classes and cases:**

    ```
    class TestSpatialAttentionBlock:
        test_output_shape         — (B*T, L, d_model) in → same out
        test_different_batch_sizes — works with B=1, B=64

    class TestTemporalAttentionBlock:
        test_output_shape         — (B, T, d_model) in → same out
        test_causal_masking       — modifying future inputs doesn't change past outputs
        test_variable_seq_len     — works with T=50, T=200

    class TestDualAttentionTransformer:
        test_output_shapes        — logits (B, n_horizons, 3), vpin (B, 1), embedding (B, d_model)
        test_without_vpin_head    — vpin_head=False: no 'vpin' key in output
        test_different_seq_lengths — T=50, T=100, T=200 all work
        test_single_horizon       — n_horizons=1 works
        test_deterministic_with_seed — fixed seed → identical outputs on same device

    class TestDeepLOB:
        test_output_shape         — (B, n_horizons, 3) from (B, T, 40)
        test_different_seq_lengths — T=50, T=100
        test_single_horizon       — n_horizons=1 works

    class TestLinearBaseline:
        test_output_shape         — (B, n_horizons, 3) from (B, T, 40)
        test_uses_last_timestep   — verify only last timestep matters (change earlier timesteps, output unchanged)

    class TestFocalLoss:
        test_gamma_zero_matches_ce     — gamma=0 equals F.cross_entropy
        test_focal_reduces_easy_loss   — gamma=2 loss <= gamma=0 loss
        test_class_weights             — weighted focal loss changes output
        test_3d_input                  — (B, n_horizons, n_classes) logits work
        test_reduction_none            — reduction='none' returns per-sample loss
        test_manual_calculation        — hand-compute focal loss for a 2-sample case, verify match

    class TestModelConsistency:
        test_all_models_same_output_shape — TLOB, DeepLOB, Linear all produce (B, 4, 3) from (B, 100, 40)

    class TestMPSCompatibility:
        test_tlob_on_mps        — forward pass on MPS device (skip if not available)
        test_deeplob_on_mps     — forward pass on MPS
        test_linear_on_mps      — forward pass on MPS
        test_focal_loss_on_mps  — loss computation on MPS
    ```

    For MPS tests, use `@pytest.mark.skipif(not torch.backends.mps.is_available(), reason="MPS not available")`.

    For the deterministic seed test:
    ```python
    torch.manual_seed(42)
    out1 = model(x)
    torch.manual_seed(42)
    out2 = model(x)
    assert torch.equal(out1['logits'], out2['logits'])
    ```
    Note: set model to eval mode for deterministic test (dropout is stochastic in train mode).

    For the manual focal loss calculation test, compute by hand for 2 samples with known logits and targets, then assert the FocalLoss output matches within 1e-5.

    Use batch_size=4 and seq_len=20 in tests (small but sufficient for shape verification). Do not use large tensors in tests.
